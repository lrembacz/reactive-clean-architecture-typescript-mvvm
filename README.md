# Reactive Clean Architecture Typescript with MVVM

This example is using several packages and patterns:
- inversify,
- rxjs,
- lit-element,
- MVVM,
- Clean Architecture,
- Reactive programming.

It's inspired by Reactive Clean Architecture approach for Android / IoS platforms.

## Building

### Building the repo

```shell
npm run build
```

### Building only types

```shell
npm run build:types
```

### Type-Checking the repo

```shell
npm run type-check
```

And to run in --watch mode:

```shell
npm run type-check:watch
```
